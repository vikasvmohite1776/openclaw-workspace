# OpenClaw Quick Start Reference

## After Computer Restart

### Step 1: Start OpenClaw Gateway (if not auto-started)
```bash
openclaw gateway start
```

### Step 2: Connect to OpenClaw
```bash
openclaw
```

Or to connect directly to this session:
```bash
openclaw chat --agent main --session agent:main:main
```

### Step 3: Verify Connection
```bash
openclaw status
```

## From iMessage
Once connected, send any message to 512-667-5576 from your phone and I'll receive it via the polling system.

## Important Files
- Context: `~/.openclaw/workspace/Context.md` (session continuity)
- Skills: `~/.openclaw/workspace/Skills.md` (capabilities reference)
- Daily Memory: `~/.openclaw/workspace/memory/YYYY-MM-DD.md`
